# Feedback repo for Wtab - iOS Style New Tab Page

Welcome to the official feedback repository for **Wtab**, the iOS-style new tab page extension for Google Chrome.

🌍 **[Download from Chrome Web Store](https://chromewebstore.google.com/detail/wtab-ios-style-new-tab-pa/iiaeekafioihefkcpadipdlkbogcacll)**

---

## 💡 What is this repository for?

This repository is dedicated to tracking **Bug Reports**, **Feature Requests**, and **General Feedback** for the Wtab Chrome Extension. 

Since the extension's source code is private, we use this public repository as an issue tracker to communicate with our users transparently. The feedback submitted through the extension's built-in "Feedback" tab is automatically posted here.

## 🐛 Found a Bug or Have a Suggestion?

We'd love to hear from you! There are two ways to submit your feedback:

### Option 1: From within the Extension (Recommended)
1. Open a new tab to view **Wtab**.
2. Click the **Settings** icon (gear icon) in the top right corner.
3. Navigate to the **Feedback** tab.
4. Fill in the title and description, then click **Submit to GitHub**. Your issue will be automatically created here.

### Option 2: Directly on GitHub
If you prefer, you can manually create an issue in this repository.
- **[Report a Bug](../../issues/new?assignees=&labels=bug&template=bug_report.md&title=%5BBug%5D+)**
- **[Request a Feature](../../issues/new?assignees=&labels=enhancement&template=feature_request.md&title=%5BFeature%5D+)**

*(Tip: When reporting a bug, please include your browser version and extension version to help us investigate faster.)*
